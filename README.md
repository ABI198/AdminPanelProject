# Admin Panel Project 
An admin panel was created for a blog application.  
<br />
<b>Technologies Used</b>
 * C#
 * LINQ
 * Asp.Net Core MVC 
 * Entity Framework 
 * .Net Core Identity
 * FluentAPI
 * AutoMapper
 * MSSQL
 * Bootstrap
 * JQuery
 * AJAX

<b>Design Principles Used</b><br>
 * MVC
 * SOLID Principles
 * Asynchronous Programming
 * N-Tier Architecture
 * Repository Pattern
 * Unit of Work Pattern
 * Data Transfer Object Pattern (Dto)
 * Result Pattern
